# Work Order Prototype (UI clone)

Matches the provided screenshots (sidebar + topbar layout, dashboard tiles, work order listing, create, execution/status, creator review, and a printable payslip/report view). Data is mocked and persisted to `localStorage`.

## Run

1. Install deps: `npm i`
2. Start dev server: `npm run dev`

## Notes

- Login/auth is not implemented (prototype).
- “Master Setting / User / Profile / SMS Config / Report” are placeholder pages except the Work Order flows.
